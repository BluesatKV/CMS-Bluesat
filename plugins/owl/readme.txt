// Installation

1. Upload the owl folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin Feedback-Owl
5. Enter a title and the database information to your feedback installation. When installed in the same database under a prefix only set the prefix.
6. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the Feedback-Owl plugin
2. Delete the owl folder in the plugins directory over FTP.
3. Done
