// Installation

1. Upload the site_editor folder into the plugins folder
3. Login into your administration panel
4. Go to Plugins/Hooks
5. Install the plugin
6. plugins/site_editor/files needs chmod 0777
7. Edit your basic site files - .htaccess and robots.txt
8. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the site_editor plugin
2. Delete the site_editor folder in the plugins directory over FTP.
4. Done
